# hello-locust

## Usage

```sh
$ locust -f locustfile.py
```

Open http://127.0.0.1:8089/ in a browser.

Enter the followings.

- Number of total users to simulate
- Spawn rate (users spawned/second)
- Host (e.g. http://www.example.com)
